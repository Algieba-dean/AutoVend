
# 丰田威兰达 双擎 2024款
## 1. 车型基本信息
### 品牌信息
- 品牌名称：丰田

- 品牌国别：德国

### 车型信息
- 车系名称：威兰达
- 具体型号：双擎
- 年款信息：2024款

以下是丰田威兰达双擎2024款的详细车型信息（基于中国市场现款车型及行业趋势预测整理，部分参数为预估，请以官方发布为准）：
