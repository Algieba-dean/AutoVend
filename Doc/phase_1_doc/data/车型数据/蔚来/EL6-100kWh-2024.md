
# 蔚来EL6 100kWh 2024款

### 车型信息
- 车系名称：EL6
- 具体型号：100kWh
- 年款信息：2024款

以下是蔚来EL6 100kWh 2024款的详细车型信息（注：蔚来官方未发布“EL6”车型，推测可能为EC6或ES6的笔误。以下信息基于ES6 100kWh 2024款整理，符合蔚来最新技术平台NT2.0的配置逻辑）：
