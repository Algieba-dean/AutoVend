# Changelog — v3.1 (2026-03-27)

## Summary

Added comprehensive **Agent Performance Evaluation Framework** — 24 dialogue scenarios across 4 categories, scored on 6 dimensions with professional RAG/Agent evaluation metrics. Established baseline KPIs and identified optimization roadmap.

## Changes

### Performance Evaluation Framework (NEW)

- **Scoring Engine** (`tests/performance/scoring.py`)
  - 6 scoring dimensions: Extraction Accuracy, Stage Correctness, Response Quality, Business Capability, Robustness, Latency
  - Per-turn `TurnScore` and per-dialogue `DialogueScorecard` dataclasses
  - Aggregate `EvaluationReport` with category breakdown
  - Weighted overall score formula (Business 35%, Response 30%, Extraction 20%, Stage 10%, Latency 5%)
  - Professional metrics: Precision/Recall/F1, conversion funnel, task completion, language consistency

- **24 Dialogue Scenarios** (`tests/performance/scenarios.py`)
  - 10 normal happy-path (diverse personas: programmer, family, luxury, budget, retired, tech, MPV, business)
  - 6 edge cases (silent user, mid-conversation correction, one-shot info dump, gradual reveal, single-dimension, budget-only)
  - 4 adversarial (off-topic, prompt injection, nonsense/emoji, mixed valid/invalid)
  - 4 bilingual (CN+EN mix, EN→CN switch, oral CN + EN brands, formal English)

- **Evaluation Harness** (`tests/performance/eval_harness.py`)
  - Automated runner with mock LLM that simulates stage-aware responses
  - Per-turn scoring: relevance, language consistency, professionalism, info gathering rate
  - JSON report persistence to `eval_results/` with timestamps
  - Human-readable console summary

- **Pytest Integration** (`tests/performance/test_performance.py`)
  - 70 test cases across 6 test classes
  - KPI target assertions per category (normal ≥ 40, edge ≥ 15, adversarial: no crash + professional)
  - Aggregate KPI tests: stage accuracy ≥ 50%, overall ≥ 30, 100% response rate
  - Auto-saves `eval_latest.json` on every run

### Documentation

- **Performance Evaluation Framework** (`docs/performance_evaluation.md`)
  - Complete KPI definitions with formulas and targets
  - Scoring rubrics for all 6 dimensions
  - Full scenario catalog (24 scenarios, 77 turns)
  - Baseline results with per-dialogue scorecards
  - Gap analysis with root causes
  - 3-phase optimization roadmap
  - Professional evaluation standards reference (RAGAS, IR metrics, NLG, dialogue systems)
